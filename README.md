# Project Summary

## Title
Palo Alto Engagement Intelligence Dashboard

## Objective
To design an executive workforce analytics dashboard that helps identify employee disengagement, burnout exposure, attrition risk, and work-life imbalance.

## Problem Statement
Organizations often react to attrition after employees have already disengaged. HR teams need an early-warning system that highlights workforce stress, low engagement, and vulnerable employee segments before resignation occurs.

## Solution
This project uses a Streamlit dashboard built on employee HR data to create:
- An Engagement Index
- A Burnout Risk Score
- Attrition monitoring views
- Department and job-role level analysis
- An action-oriented review panel

## Key Metrics
- Engagement Index
- Work-Life Balance
- Attrition Rate
- Burnout Risk Level
- Department-level risk concentration

## Tools Used
- Python
- Pandas
- Streamlit
- Plotly

## Outcome
The final dashboard presents HR insights in an executive-friendly format and helps decision-makers prioritize intervention areas based on data.

## Future Enhancements
- Add time-series trend tracking
- Add predictive attrition modeling
- Add manager-level drilldown
- Add employee segmentation by tenure and compensation
