# Project Summary

## Title
Palo Alto Engagement Intelligence Dashboard

## Objective
To build an executive-style workforce analytics dashboard that helps detect disengagement, burnout exposure, and attrition risk using employee HR data.

## Problem Statement
Traditional HR reports often describe what has already happened. Organizations need a more proactive way to identify early warning signs of employee dissatisfaction and burnout before attrition occurs.

## Solution
This Streamlit dashboard combines HR indicators into an interactive decision-support system using:
- Engagement Index
- Burnout Risk Score
- Attrition analysis
- Department and role segmentation
- Action Center for priority review

## Tools Used
- Python
- Pandas
- Plotly
- Streamlit

## Key Outcome
The final dashboard presents workforce risk signals in a professional executive format and supports faster HR intervention decisions.

## Future Scope
- Add predictive attrition modeling
- Add time-based trend analysis
- Add manager-level drilldowns
- Add employee segmentation by tenure and compensation
